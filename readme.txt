我的自画像

大家好，我叫戴林君，今年9岁，我是个开朗的小女孩。
	你看我，小个子，笑眯眯的，可爱不可爱？奶奶老嫌我瘦，让我多吃肉，可是我不喜欢吃肉，我对奶奶说：“别看我瘦，连舅舅都搬不动的铁条我都搬得动，可强壮了！”
	我是班级中的“小矮人”，集队时总是排在前面几个，我上三年级了，不过我的身高才1.28米，体重才20千克，真担心大一点的风就能把我刮倒。我的牙齿最讨厌了，特别是大门牙，换了牙之后一直在不停的长，现在都像块小石头了。每次看着镜中的大门牙，我就郁闷。我有一双大大的丹凤眼。我这大大的眼睛，都是用来盯着老师的呢！
	我是个不太爱笑的小女孩，整天都不笑，老师讲笑话，同学们都笑了，只有我没笑。在学校里，我努力学习；在家里，我认真完成爸爸妈妈交给我的任务。
	我喜欢骑自行车。我每个星期六和星期天都会去骑自行车。我还喜欢画画，我还得过三等奖呢！
	你们喜欢我吗？
																																	2005年10月8日
